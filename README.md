## Coding Style
### Naming
- Class Name: UpperCamelCase
- Method Name: lowerCamelCase
- Non-Final Variable: lowerCamelCase
- Final Variable: SCREAMING_SNAKE_CASE
- Declaration
  + One Declaration in One Line
    * int x; /* Good */
    * int x, y; /* Bad */
  + Array: Type[] name
